CAN Based Automotive Dashboard

Name:Rosna Shaji

Batch Number:25040B_014

Project Description

This project implements a CAN (Controller Area Network) based Automotive Dashboard using PIC18F4580 microcontrollers. The system consists of three Electronic Control Units (ECUs) communicating over the CAN bus to display RPM, Speed, Gear Position, and Indicator status on a 16x2 Character LCD in real time.

Features
- Real-time CAN communication between multiple ECUs
- RPM display
- Speed display
- Gear position display
- Indicator status display (Left, Right, Hazard, Off)

Hardware Used
- PIC18F4580 Microcontroller
- 16x2 Character LCD
- Potentiometer
- Digital Keypad

Software Used
- MPLAB X IDE
- XC8 Compiler
- Embedded C
